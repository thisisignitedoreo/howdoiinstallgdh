How do I install GDH?

Important notice

As you know the GDH is currently broken and does not work with newer GD 2.2 versions (2.201 and later), so, I'll guide you through downgrading process. TBD: update when fix is finally out

Downgrading

I'll be using pirated build of the game by toby himself, because:

  1. I don't have GD on steam
  2. I don't know how to use steam depots

Okay, lets install old GD:

  1. Download old GD 2.2
  2. Unpack it wherever you want
  3. Open it from GeometryDash.exe file or create shortcut to this file

Installing

  1. Download latest GDH (Release.zip)
  2. Unpack it into GD folder, overwritting libExtensions.dll

Voila!
